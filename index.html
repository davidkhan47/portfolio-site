Steps Create Project

/001Speak

At this stage, a conversation is held with the customer in order to determine which site needs to be created, the target audience is determined, and then a detailed technical task is created!

/002Create Design

Based on the terms of reference, a website design is created. The prototype is created first, then the main design. The finished layout is transferred to the client and awaits his approval. The design will be finalized until until it is approved!

/003Create Site

After successful confirmation of the design, the site layout begins. The layout is transferred from the layout (picture) to the working site and become dynamic. Fonts, CSS styles, JS libraries, cross-browser settings are connected to the resource and adding a responsive version of the site!

/004Testing

After a successful layout, testing begins. At this stage, functionality is monitored site to identify errors, which are subsequently eliminated, and then the finished project is transferred to the customer!
